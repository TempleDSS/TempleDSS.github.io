# TempleDSS.github.io

Work-in-progress site that takes a "collections as data" approach to the [Loretta C. Duckworth Scholars Studio blog](https://sites.temple.edu/tudsc/). We scraped the full blog archive and turned it into charts, tables, and a searchable project index -- treating a decade of blog posts as a dataset rather than just a website.

Built with [Eleventy](https://www.11ty.dev/). Still evolving.
